Accessibility

This website follows the W3C Accesibility guidelines. To ensure the our website is accessible we made sure that users can fully navigate the website with just a keyboard. With the one exception of our interactive game, which we apologise for any inconvience this has caused. We have also ensured that our website works with screen readers by using alternative text on all our images and having descriptive markup.

Should any user have difficulty in accessing the content of this site, please contact our web team and we will try ammend the problem as soon as possible.
